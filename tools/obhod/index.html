Обходы

Чеклист проектов для «обхода»: карточки с тегами, описание, участники и команды телепортации по пунктам. Данные хранятся в localStorage этого сайта; можно импортировать и экспортировать список в JSON. Двойной клик по карточке отмечает просмотр; двойной клик по полосе прогресса открывает случайный непросмотренный обход.
