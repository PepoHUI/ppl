DependecyChecker

Вставьте ссылку на мод с Modrinth. Утилита запрашивает официальный API из вашего браузера: подбирая зависимости под версию Minecraft и модлоудер, затем выводятся зависимости с типом, ссылками на страницу и прямой загрузкой .jar, если Modrinth отдаёт файл.

Modrinth и GitHub Pages

  • В части браузеров система может подменить User-Agent — при ошибках 403 попробуйте другой браузер или запуск с локальной машины.
  • Лимит: до 300 запросов в минуту с одного IP.
  • Точность: если в ссылке указана конкретная версия на сайте, используется она; иначе — самая новая сборка, подходящая под выбранные версию игры и загрузчики.

Если пусто, Modrinth вернёт последние сборки только по загрузчикам.
